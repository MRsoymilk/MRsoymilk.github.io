Ubuntu 18.04 useradd无法创建用户主目录

使用 useradd 添加用户，-d 指定用户家目录，/home 目录下并没有创建用户 🏠 目录

解决办法

方法一：在/etc/login.defs文件中添加或修改为CREATE_HOME yes

方法二：useradd使用-m参数,例如useradd -m apple
