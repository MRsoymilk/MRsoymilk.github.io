配置网络便于连接

修改/etc/dhcpcd.conf

主Raspberry

interface eth0
static ip_address=192.168.0.1/24

从Raspberry

interface eth0
static ip_address=192.168.0.233/24

重启设备后两台设备的ip分别为192.168.0.1和192.168.0.233

即可通过网线交互。

例如，在192.168.0.1的Raspberry上：

ping 192.168.0.233
