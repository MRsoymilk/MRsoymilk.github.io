使用platform命令行操作

编译程序：

platformio run

上传程序：

platformio run --target upload

打开串口监视器：

pio device monitor

platformio配置文件操作

[env:esp32cam]
platform = espressif32
board = esp32cam
framework = arduino
lib_deps = roboticsbrno/ServoESP32@^1.0.3
upload_port = /dev/ttyUSB0
monitor_speed = 9600
