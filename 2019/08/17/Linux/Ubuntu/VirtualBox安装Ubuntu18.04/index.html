Encypt the new Ubuntu install for security： 加密安装，好像并不需要。

Use LVM with the new Ubuntu installation：LVM(Logical Volume Manager，逻辑卷管理其)，Linux 下对磁盘分区进行管理的一种机制。可以在磁盘不用重新分区的情况下动态调整分区大小。
