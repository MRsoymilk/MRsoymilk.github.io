写在前面

JHipster 命令行创建的示例项目，使用的 Maven 包管理工具
参考：
Spring Boot 打 Jar 包部署
jHipster - 项目启动发布

步骤

进入 pom.xml 所在目录中

JHipster 已经帮我们做好了绝大部分工作
如果是普通 Spring Boot 项目
添加 Maven 依赖插件

<build>
    <plugins>
        <plugin>
            <groupId>组织名.项目名.模块</groupId>
            <artifactId>spring-boot-maven-plugin</artifactId>
        </plugin>
    </plugins>
</build>

完全参考博客

运行 Maven 命令

mvn clean package -Dmaven.test.skip=true

运行文件

打包完成后进入target目录, 执行：
java -jar your_project.jar
此时可以认为部署在此台主机

完成

浏览器输入对应 ip+端口 访问即可
